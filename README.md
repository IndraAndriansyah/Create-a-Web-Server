# A387-Jarkom-Labs

Untuk menjalankan project ini, pastikan `npm` sudah terinstall pada komputer/laptop Anda.
Git Clone: https://github.com/dicodingacademy/a387-jarkom-labs.git

---

Tata cara menjalankan project:

1. Install node modules

```
npm install
```

2. Jalankan project

```
npm run start
```
